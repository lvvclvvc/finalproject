1. 程式的原理與功能
本程式旨在模擬流體流經平整板面時，其動量邊界層與熱邊界層的成長過程。透過數值模擬，將抽象的流體力學理論轉化為動態圖表，協助觀察物理參數對流場的影響。

物理與化學原理
* 雷諾數 (Reynolds Number, Re)：定義慣性力與黏滯力的比值，決定了邊界層的成長速率與流態特徵。
* 普蘭特數 (Prandtl Number, Pr）：定義動量擴散與熱擴散的比值。在化工傳遞程序中，這決定了流體速度分布與溫度分布的類比關係。

數學方程式
本程式同時求解非線性耦合常微分方程式：
1. 動量方程式 (Blasius Equation)：
   f''' + 1/2 f f'' = 0
   其中 f 為流函數之無因次量，邊界條件為：f(0)=0, f'(0)=0, f'(infinity)=1。
2. 能量方程式 (Thermal Energy Equation)：
   theta'' + Pr/2 f theta' = 0
   其中 theta為無因次溫度，邊界條件為：theta(0)=0 (壁溫), theta(infinity)=1 (主流溫度)。
3. 理論邊界層厚度:
    動量邊界層厚度 delta approx 4.91 sqrt((nu x)/(U))
    熱邊界層厚度 delta_t approx delta Pr^-1/3

## 2. 使用方式
1. 環境配置：使用者需先安裝 Python 3 及其必要套件 `numpy`, `matplotlib`, `scipy`。
2. 執行程式：啟動 `finalproject.py` 後將彈出互動視窗。
3. 參數調整：
   拖動 Velocity U 滑桿：觀察流速增加時動量邊界層變薄的趨勢。
   拖動 Viscosity nu 滑桿：觀察黏度增加時邊界層增厚的現象。
   拖動 Prandtl Pr 滑桿：觀察熱邊界層相對於動量邊界層的縮減或擴張。

## 3. 程式的架構
本程式架構嚴格對應物理與數學原理，分為以下模組：
 數值核心層：呼叫 `scipy.integrate.solve_bvp` 求解耦合微分方程式，將物理邊界值問題轉為無因次數值解。
 理映射層：將無因次解 eta, f', theta 根據使用者輸入的實際物理量 U, nu, Pr 映射回具備物理意義的空間座標 y 與速度 u。
 繪圖顯示層：使用 `matplotlib` 建立雙子圖佈局，左圖呈現剖面分佈，右圖呈現隨距離 x 的空間成長。
 互動控制層：配置 `Slider` 物件監聽輸入變動，並即時觸發繪圖更新函數。

## 4. 開發過程
 在開發過程中，主要遭遇以下技術困難與解決方案：
 數值收斂挑戰：在求解包含 $Pr$ 數的能量方程式時，初始網格若過於稀疏會導致計算發散。透過調整 `solve_bvp` 的初始猜測矩陣與網格解析度成功解決。
 互動元件穩定性：初期拉桿在操作數次後會發生功能凍結。經分析為 Python 垃圾回收機制回收了局部變數，解決方法為將拉桿物件儲存於全域清單中以維持引用狀態。
