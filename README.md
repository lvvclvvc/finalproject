這是我在計算機程式課程中的期末專案。這個工具透過 Python 數值模擬，將流體力學中抽象的「邊界層（Boundary Layer）」理論轉化為直觀的互動圖表。

1. 程式功能與技術原理
核心功能
本程式能即時模擬流體流經平整板面時的速度分布變化。透過拉桿調整參數，可以直接觀察雷諾數 (Re)如何影響邊界層的厚度與形狀。

技術原理
* 物理背景：基於層流邊界層理論，求解的Blasius方程式。
* 數值解法：由於該方程式為非線性常微分方程式（ODE），本程式使用 `scipy.integrate.solve_bvp`（邊界值問題求解器）進行數值運算，求得無因次的速度剖面。
* 核心方程式：
    f''' + \frac{1}{2} f f'' = 0
* 理論厚度：定義速度達到主流 99% 處為邊界層厚度delta，其理論近似公式為：
    \delta \approx 4.91 \sqrt{\frac{\nu x}{U}}

2. 使用方式

1.  環境準備：請先確保電腦已安裝 `numpy`, `matplotlib`, `scipy` 這三個 Python 套件。
2.  執行程式：執行 `finalproject.py` 後會彈出一個互動視窗。
3.  調整參數：
    * Velocity U：調整主流速度。觀察速度變快時，邊界層如何變薄。
    * Viscosity nu：調整運動黏度。觀察黏性增加時，動量傳遞如何讓邊界層變厚。
4.  圖表觀察：左圖是速度剖面（紅線），可以看到流速從板面為 0 增加到主流速度的過程；右圖是邊界層沿著x方向發展的成長趨勢（藍色區塊）。

---

3. 程式架構
本程式採用模組化設計，結構如下：
* 物理運算層：利用 `blasius_ode` 定義微分方程式，並透過 `solve_bvp` 取得數值解。
* 視覺化介面：使用 Matplotlib 設定雙子圖佈局，並將 $y$ 軸範圍固定在 0 到 0.08，確保視覺觀察的一致性。
* 互動邏輯層：由 `update` 函數負責重新計算物理座標，並在參數變動時同步重繪畫布。
* 控制元件：配置 `Slider` 監聽滑鼠動作，即時反饋參數變化的結果。
